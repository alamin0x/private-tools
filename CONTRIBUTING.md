# Contributing to PrivateTools

First off, thank you for considering contributing to PrivateTools! It's people like you that make it such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Check the issues tab to see if it has already been reported.
- Use a clear and descriptive title.
- Describe the exact steps which reproduce the problem.

### Suggesting Enhancements
- Check the issues tab to see if it has already been suggested.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the existing style.

## Style Guide
- Use TypeScript for all new components.
- Follow the existing project structure in `src/tools/`.
- Use CSS variables for colors to maintain Dark Mode support.
